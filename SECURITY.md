# Security Policy

## Reporting a Vulnerability

We take the security of BrajPath seriously. If you find a security vulnerability, please do not open a public issue. Instead, please report it via one of the following methods:

- **Email**: security@brajpath.example.com (Placeholder)
- **GitHub**: Report via the [GitHub Security Advisory](https://github.com/Mradulmanimishra/BrajPath/security/advisories/new) feature.

## Supported Versions

We only support the latest version of BrajPath on the `main` branch.

## Disclosure Process

1. We will acknowledge receipt within 48 hours.
2. We will provide an estimated timeline for a fix.
3. Once fixed, we will publish a security advisory.
